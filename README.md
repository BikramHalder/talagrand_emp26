# Talagrand's Concetration Inequality and its Applications to Empirical Risk Minimization: Simulation Code and Data for the Course Project
This repository contains [tex files](tex_files) and [R code](code) for the "Empirical Processes and Talagrand's Inequality" course project, offered by Prof. Airjit Chakrabarti in the Spring 2026 at the Indian Statistical Institute, Kolkata. The tex files contain a detailed report on Talagrand's concentration inequality and its applications to empirical risk minimization based on Koltchinskii's notes[^1], while the R code includes original simulations by [Eshan De](@eshanay) that illustrate the theoretical results discussed in the report.

[^1]: Vladimir Koltchinskii. "Oracle Inequalities in Empirical Risk Minimization and Sparse Recovery Problems". In: Springer Lecture Notes in Mathematics (2011). École d’Été de Probabilités de Saint-Flour XXXVIII-2008. doi: [10.1007/978-3-642-22147-7](https://doi.org/10.1007/978-3-642-22147-7).
